Contains a minimal build of OpenCV 4.13 for PS Move tracking.

This includes:
- OpenCV binaries and headers, licensed under Apache License 2.0 (see LICENSE-OpenCV.txt).
- Build scripts and packaging files authored for this repository, licensed under GPL-2.0 (see LICENSE).

How to update to new version:
- Change ref to version tag in opencv.yml
- Push change and trigger manual workflow in github actions
- Download artifact and commit new files to this repository
